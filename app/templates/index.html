The Safecast API

Query and add to the Safecast dataset with your own application.

API Endpoint

GET https://api.safecast.org/.json

Available Resources

  • Users Add and view user accounts
  • Measurements Add and view measurements
  • bGeigie Imports Add and view bGeigie Imports
  • Devices Add and view Devices

Users

  • Get a list of users GET /users.json
  • Add a new user POST /users.json
  • View a user GET /users/1.json

Measurements

  • Get a list of measurements GET /measurements.json
  • Add a new measurement POST /measurements.json
  • View a measurement GET /measurements/13192557.json

bGeigie Imports

  • Get a list of bGeigie imports GET /bgeigie_imports.json
  • Add a new bGeigie import POST /bgeigie_imports.json
  • View a bGeigie import GET /bgeigie_imports/10757.json

Devices

  • Get a list of devices GET /devices.json
  • Add a new device POST /devices.json
  • View a device GET /devices/4.json

Using the API

Measurements

The Measurement is the basic unit in the Safecast database.

A measurement is composed of four key parts.

  • Timestamp captured_at
  • Unit unit
  • Value value
  • Geopoint location (latitude, longitude)
